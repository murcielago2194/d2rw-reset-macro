# D2RW RESET MACRO

### Macro for automating Diablo 2 Resurrected (Reign of Warlock) speedrunning resets

## Basics
- Based on the D2R (LOD) reset macro
- Warlock support
- UI-based configuration
- New and safety features
- Code optimizations

## Changelog

### v1.0.0 (Changes compared to the LOD version)

- Warlock updates (character creation screen)
    - Added Warlock class selection
    - Added version selection
    - Updated hardcore icon position

- New features
    - UI for hotkey and character configuration
    - Hotkey to bring the UI to the foreground
    - Character list filtered by selected version
    - Ctrl / Shift / Alt modifier support for hotkeys
    - Automatic delay handling on loading screens

- Misc
    - Only one reset macro can run at a time
    - Reset hotkey only works when D2R is active
    - Removed delay parameters
    - Removed character deletion
